# One People Mobile Mockup

Project ini sekarang menjadi aplikasi iOS SwiftUI bergaya OnePeople/HRIS untuk kebutuhan employee self-service.

Fitur utama:

- Dashboard One People dengan status absensi harian
- Attendance history dan pencarian riwayat
- Leave info, approval inbox, dan payroll ringkas
- Personal info dan employment profile
- Data mock lokal agar mudah dipreview di Xcode

Cara menjalankan:

1. Buka `OnePeople.xcodeproj` di Xcode.
2. Pilih target `One People`.
3. Atur `Signing & Capabilities` bila perlu.
4. Jalankan ke iPhone Simulator atau device fisik.

Referensi desain modul:

- One People App Store listing menampilkan fitur seperti `Personal Info`, `Attendance`, `Permission`, `Leave Info`, `Change Schedule`, `Adjust Presence`, `Claim`, dan update terbaru menambah `Reservation` serta `Daily Report Subordinate`.
